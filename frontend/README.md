# Frontend

Frontend React + TypeScript + Vite da plataforma de marketplace para aulas práticas de direção.

## Main Screens

- Home with instructor search and booking
- Login and register
- Become instructor
- Central do Instrutor
- Meus Agendamentos

## Current UI Flows

- Students can search instructors and request one or multiple lesson slots
- Instructor booking uses date-range search, day selection and multi-slot selection
- Instructors configure availability by period, weekdays and multiple time ranges
- Overlapping instructor availability is merged on save
- The instructor central shows pending requests, confirmed lessons, availability, historical lessons, earnings, stats and reviews

## Build

```bash
npm install
npm run dev
npm run build
```
